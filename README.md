# Operational management system for sports facilities

System designed for managing reservations, clients, payments and reports for a sports complex.

### Main idea:
Digitize and organize the management of shifts and resources in sports facilities.


### Technologies used:
- Python
- Tkinter
- SQLite

  
### Architectural aspects:
- Monolithic MVC architecture
- Application of the DAO pattern
- Separation between business logic and persistence
- Use case-based design
- Pre-modeling using diagrams
